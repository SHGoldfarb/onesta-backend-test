# onesta-backend-test

## Requires

```
node: 22.x
yarn: ^1.22
```

## Install dependencies

```
yarn
```

## Run dev server

```
yarn dev
```

## Run tests

```
yarn test
```

#### Run tests in watch mode

```
yarn test:watch
```
